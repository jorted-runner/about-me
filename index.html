Chefs Cabinet

Chefs Cabinet is a dynamic recipe-sharing platform that lets users share, rate, and review recipes while following their favorite chefs. It leverages AI to generate custom recipes, images, and personalized shopping lists, enhancing the cooking experience for each user. Built with Python and Flask, the backend integrates an SQLite relational database to manage user profiles, recipes, and cookbooks, and is deployed on AWS for performance and scalability.
